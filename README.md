# Лабораторная работа №10 — PySpark и анализ данных

## Описание проекта
Цель проекта — познакомиться с основными возможностями обработки данных в
распределённой среде с использованием **Apache Spark (PySpark)**.  
В рамках работы выполняется загрузка датасета о продажах видеоигр,
проведение первичного анализа данных и вывод основных статистических показателей.

## Используемые инструменты
- Python 3.x
- PySpark
- Jupyter Notebook / Python script
- CSV-датасет `vgsales.csv`

## Структура проекта
│ vgsales.csv — исходный датасет

│ pyspark_lab10.ipynb — основной скрипт с анализом данных

## Источник данных
Датасет взят из открытого репозитория Kaggle:
- Тематика: мировые продажи видеоигр
- Файл: `vgsales.csv`
- Ссылка: https://www.kaggle.com/datasets/dandanjia/vgsales-csv

Размер исходного датасета:
- 16599 строк
- 11 столбцов
- колонки: Rank, Name, Platform, Year, Genre, Publisher, NA_Sales, EU_Sales, JP_Sales, Other_Sales, Global_Sales
## Как запустить проект

### Шаг 1 — Установить зависимости
```pip install pyspark```

### Шаг 2 — Запустить PySpark-код
```pyspark_lab10.ipynb```
